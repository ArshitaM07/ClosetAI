# 👕 ClosetAI

ClosetAI is an AI-powered wardrobe assistant that helps users organize clothing, upload outfits, and receive personalized styling suggestions.

The goal is to make choosing outfits easier by combining wardrobe management with AI recommendations.

---

## ✨ Features

- 💬 AI-powered chat assistant
- 📸 Smart wardrobe scanner
- 👕 Upload clothing items
- 🖼️ Visual wardrobe gallery with clothing thumbnails
- 🌤️ Weather-based outfit suggestions
- 👤 Personal style profile
- ❤️ Personalized recommendations
- 🌙 Modern dark UI

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/ArshitaM07/ClosetAI.git
```

Move into project directory:

```bash
cd ClosetAI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run application:

```bash
python app.py
```

Open browser:

```text
http://127.0.0.1:5000
```
